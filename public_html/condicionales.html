Ejemplo de uso de Switch

var day;
switch(new Date().getDay()){
case 0:
day = "Domingo";
break;
case 1:
day = "Lunes";
break;
case 2:
day = "Martes";
break;
case 3:
day = "Miercoles";
break;
case 4:
day = "Jueves";
break;
case 5:
day = "Viernes";
break;
case 6:
day = "Sabado";
break;

Al igual que en otros lenguajes, tambien podemos usar default para manejar las excepciones
